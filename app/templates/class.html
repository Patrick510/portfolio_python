Aula C/Atividade

PRIMEIRO CONTATO COM FLASK - EXERCICIOS PARA FIXAÇÃO

Toda programação em web trabalha com Requisição e Resposta.
O backend utiliza uma programação reativa, ou seja, não inicia com processos em andamento, conectando Cliente e Servidor.
O Flask permite que nosso código Python fique esperando as suas chamadas.

Pesquisar:
Existe um tipo de variável especial (a mais comum) em que o nome da variável e seu valor são iguais, como:
- __name__
- __root__

Rodando localmente:
Transformamos nossas máquinas em servidores.
Exemplo:
if __name__ == '__main__':
app.run(debug=True)
- Este trecho faz com que o código rode sem parar e fique aguardando ordens, funcionando como um servidor.
- debug=True: Permite editar o código com atualizações automáticas.
- port=3030: Configura uma porta específica.
- host='0.0.0.0': Permite que todos na mesma rede acessem seu código.

Anotações:
- Começam com @.
- Exemplo: @app.route define a rota desejada, sendo ('/') a raiz do projeto.

Tarefa:
Criar uma segunda rota de contato e adicionar dados de contato.
Fazer, daqui 4 semanas, um protótipo do trabalho.

Atividades:
- Atividade 01:
Crie uma página web (rota) com a tag canvas que permita ao usuário deslocar uma fotografia para a esquerda e para a direita usando as setas do teclado.

- Atividade 02:
Crie uma rota que permita ao usuário capturar uma fotografia da webcam e exibi-la.

- Atividade 03:
Crie uma rota com uma página web que exiba uma tabela sem usar a tag de tabela. Deve conter:
- 997 linhas
- 5 colunas: id, nome, sobrenome, email, ações.

- Atividade 04:
Crie uma rota com 3 links, um para cada atividade, em versões estilizadas.

- Atividade 05:
Crie 6 rotas, cada uma estilizada, e inclua o currículo de um integrante do grupo em cada rota.
